# Join With an Invite Code

Use this page if you were given:

- a Discord invite link
- a 6-digit access code

## Steps

1. Join the Discord server using the invite link you were given.
2. After you join, go to a channel where bot commands are allowed.
3. Run the slash command:

```text
/enter_role
```

4. A popup will appear asking for your 6-digit code.
5. Enter the code exactly as you received it.
6. Submit the form.
7. If the code is valid, the bot will assign your role automatically.

## What To Expect

- valid code:
  - your mapped role is assigned
- code already used by you:
  - the bot tells you that you already have the role
- invalid or expired code:
  - the bot will reject it and you will need a new code or help from a moderator/admin

## Important Notes

- The code is expected to be 6 digits.
- Enter it exactly as provided.
- Do not share the code unless the person who issued it told you it can be reused.
- If the role is not assigned after a valid submission, a server admin may need to check the role mapping or bot permissions.

## Troubleshooting

### I joined the server but do not know what to do next

Run:

```text
/enter_role
```

Then enter your 6-digit code in the popup.

### The code does not work

Possible reasons:

- the code was entered incorrectly
- the code is no longer active
- the mapped role was changed or removed

If that happens, contact the person or staff member who gave you the code.

### I submitted the code but did not get the role

Possible reasons:

- you already had the role
- the bot does not currently have permission to assign that role
- the mapping needs to be corrected by an admin

Contact a server admin or moderator and tell them:

- you joined successfully
- you used `/enter_role`
- the 6-digit code did not assign the role

## Short Version

Join the server with your invite link, run `/enter_role`, enter your 6-digit code, and submit it. If the code is valid, your role will be added automatically.
